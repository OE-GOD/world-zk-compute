What I Built

I built MOOUSER from scratch as a solo developer — the ZK prover, circuit compiler, on-chain Solidity verifiers, Stylus WASM verifier (24KB), REST APIs, 4 SDKs, CLI tool, and full production infrastructure.


The system generates zero-knowledge proofs that an ML model (XGBoost, LightGBM, Random Forest, Logistic Regression, Neural Networks) produced a specific output on specific inputs. Verification takes 629ms off-chain or 3-6M gas on-chain.


The architecture uses a TEE happy path ($0.0001/inference) with ZK dispute fallback — you only generate expensive proofs when someone challenges a result. This makes cryptographic accountability economically viable.


Open source. Apache-2.0. Cryptographic infrastructure that banks depend on needs to be auditable.

Why This Matters

EU AI Act enforcement begins 2026. US regulators are increasing scrutiny on ML-driven lending. Banks need to prove their AI makes correct decisions. Today the proof is log files. Log files can be edited. Cryptographic proofs cannot.
