The Problem

AI makes decisions. Nobody can verify them.

Credit approvals, fraud scores, insurance risk assessments — regulated institutions deploy ML models that affect millions of people. Regulators, auditors, and counterparties have no way to confirm that the declared model actually ran, that the input data was not tampered with, or that the output was not fabricated.

The Solution

Cryptographic proof that the right model ran on the right data and produced the right output.

World ZK Compute generates a zero-knowledge proof of ML inference. The proof is non-interactive, publicly verifiable, and can be checked on-chain or off-chain in milliseconds. The model weights and input data stay private.

How It Works

1

Run Inference

Execute your ML model (XGBoost, LightGBM, Random Forest, Logistic Regression) inside a TEE or the ZK prover. The computation is recorded as an arithmetic circuit.

2

Generate Proof

The GKR+Hyrax protocol produces a cryptographic proof that the circuit was evaluated correctly. A Groth16 wrapper compresses EC checks for efficient on-chain verification.

3

Verify Anywhere

Check the proof on Ethereum, Arbitrum, in a browser, or via REST API. Verification is fast (<1s off-chain, ~3-6M gas on-chain) and requires no trust in the prover.

See It In Action

Verify an XGBoost credit scoring proof in your browser. The WASM verifier runs the same GKR+Hyrax cryptographic checks as the on-chain Solidity contract. No backend required.

Use Cases

Credit Scoring

Prove to regulators that the approved model produced each credit decision. Audit trail without exposing proprietary model weights.

Fraud Detection

Demonstrate to counterparties that transaction risk scores were computed by the declared model, not overridden manually.

Insurance Claims

Verify that automated claims decisions followed the registered model. Reduce dispute resolution from weeks to seconds.

AML Compliance

Provide cryptographic evidence that AML screening models ran on the correct data. Satisfy cross-border regulatory requirements without sharing raw data.

Get Started

Install the verifier SDK in your language of choice.

Python pip install zkml-verifier
JavaScript npm install @worldzk/verifier
Rust cargo add zkml-verifier

Or run the full stack with Docker:

docker compose -f docker-compose.bank-demo.yml up -d

Architecture

TEE Happy Path

AWS Nitro Enclave runs inference and signs the result. Cost: ~$0.0001 per inference. Latency: milliseconds.

ZK Dispute Resolution

If challenged, a GKR+Hyrax proof settles the dispute on-chain. Groth16 wrapper keeps gas under 6M on Arbitrum Stylus.

On-Chain Verifier

Solidity + Stylus (WASM) contracts on Ethereum and Arbitrum. Upgradeable via UUPS proxy. Pausable for emergency response.
