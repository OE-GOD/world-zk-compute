How It Works

TEE Attestation

Run inference in a Trusted Execution Environment. The enclave signs the result with an ECDSA attestation — no one can tamper with the output.

POST /infer → attestation

ZK Dispute Resolution

If a result is challenged, prove correctness with a GKR+Hyrax zero-knowledge proof. The on-chain verifier settles the dispute in one transaction.

~3-6M gas on Arbitrum

Multi-Model Support

XGBoost, LightGBM, Random Forest, and Logistic Regression. Load any scikit-learn or XGBoost model and prove its inference.

--model-format auto

Off-Chain Verification

Verify proofs without a blockchain. Use the REST API, Python, JavaScript, or Rust SDK. Batch verify hundreds of proofs per second.

POST /verify → {verified: true}
